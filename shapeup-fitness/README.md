# рЯПЛпЄПвАНвЩАпЄП Shape Up Fitness

A gym website made in React (TypeScript) with Google's Firebase as a backend.

Check out the live [demo](https://shapeupfitness-dev.netlify.app)

## Notable features:

- Login/register an account
- Unique QR code for checking current active membership data (you can just scan it with your phone)
